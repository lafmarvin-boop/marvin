C'est un assistant de coding IA qui transforme un dossier de code, docs ou images en un **graphe de connaissances queryable**. Commandes principales :

- `graphify install` — installe la skill dans Claude Code, Cursor, Codex, etc.
- `graphify path "A" "B"` — trouve le chemin le plus court entre deux nœuds du graphe
- `graphify explain "X"` — explique un nœud et ses voisins
- `graphify query "<question>"` — interroge le graphe en BFS/DFS
- `graphify watch <path>` — surveille un dossier et rebuild le graphe automatiquement

Pour installer la skill dans Claude Code : `graphify install --platform claude`
